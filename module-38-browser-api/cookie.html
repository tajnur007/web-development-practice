What is Cookie?

Cookie is a set of texts like username and password which is stored in user's browser. Cookies are usally used for identifying specific user and improving user experience. Data sorted in a cookie is created by the server upon your connection. This data is labeled by an ID unique to you and your computer. When cookies are exchanged between your computer and server, the server reads the ID and knows what information need to serve you.

You can chech your stored cookies in your browser. Goto Application tab of your browser and find cookie section.

You can console cookies using 'document.cookie' command. To see some operations regrding cookie, see the scripts of this page with console of your browser.
