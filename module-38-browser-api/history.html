History API

To read details about History API, please visit this page.
Practice below properties in browser console.

Properties

"window.history" is same as "history". If you want to verify it then goto browser console and write window.history == history. Then it will return "true".

Basically "history" has following three operations:

  • history.back()
  • history.forward()
  • history.go()

history.back()

It will act as a user clicked on back button of the browser.

history.forward()

It will act as a user clicked on forward button of the browser.

history.go()

history.go(-1) will bring user one step back page. history.go(-5) will bring user five steps back page. history.go(3) will bring user three steps forward page if such pages are exits. But history.go(0) or history.go() will refresh the current page.
